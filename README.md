#  Cinematic Hero Slider

A modern full-screen animated hero slider built with pure HTML and CSS.

This project demonstrates advanced layout techniques, layering, typography styling, and animation effects without using JavaScript.

<br>

###  Features

 Fullscreen hero section (100vh layout)
 Smooth animation effects
 Layered image composition
 Clean typography design
 Navigation between slides
 Built with pure HTML & CSS (No JavaScript)  

<br>

###  Technologies Used

 HTML5  
 CSS3  
 Flexbox  
 CSS Animations  
 Positioning & Layering
